---
glyph: '⌹'
pattern: 'r←⌹y'
name: Matrix Inverse
---

Applies to matrices of `y`. If `y` is a matrix, `r` is the left inverse of the matrix which satisfies the property that `r+⍆∙×y` (the matrix product of `r` and `y`) is an identity matrix. If `y` is a vector, `r` is `∨⌹⍉∧y`. If `y` is a scalar, `r` is `÷y`.
